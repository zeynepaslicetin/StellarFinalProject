# 🌟 Stellar Wish Wall dApp

A decentralized application (dApp) built on the Stellar Network that allows users to connect their wallets, view their XLM balances, and post messages to a digital "Wish Wall".

## 📝 Project Description
This project demonstrates the integration between a Python-based backend (FastAPI) and a frontend interacting with the Stellar Blockchain via the Freighter wallet. It serves as a foundational example of how to fetch blockchain data and manage a simple off-chain data store (the Wish Wall).

## 🚀 Features
- **Stellar Testnet Integration:** Operates on the official Stellar Test network.
- **Freighter Wallet Support:** Securely connect and retrieve public keys.
- **Real-time Balance Inquiry:** Fetches native XLM balances using Stellar Horizon API.
- **Wish Wall:** A custom API endpoint to post and list user messages.

## 🛠️ Tech Stack
- **Frontend:** HTML5, JavaScript (ES6+), Axios.
- **Backend:** Python 3.x, FastAPI, Uvicorn.
- **Blockchain:** Stellar SDK, Freighter Wallet API.
